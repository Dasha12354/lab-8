# Лабораторная работа №8. Введение в React + JSX

Простое To-Do приложение на React с сохранением задач.


## Что такое компоненты, props и state?
- **Компонент** — переиспользуемый кусок интерфейса (App, ToDoItem и т.д.)
- **Props** — данные, которые родитель передаёт дочернему компоненту (например, `addTask`, `tasks`)
- **State** — изменяемые данные внутри компонента (массив `tasks`)

## Какую часть написал ИИ?
Весь базовый код был сгенерирован ИИ по промпту из методички, я его собрал, доработал стилями, добавил localStorage и кнопку "Очистить всё".

## Что я понял про компонентный подход
Теперь вместо того чтобы думать "как изменить DOM", я думаю "какие данные у меня есть → как должен выглядеть интерфейс при этих данных". React сам обновляет только нужные части страницы — это очень удобно и меньше багов.
